    Describe the difference between C# and .NET Framework in 2-3 sentences.
    Write your description in a text file called “csharp-and-dot-net-framework.txt”. Use English.

Solutin 13.:
--------------
C# is a programming language, .NET is a term that covers both the .NET Framework (an application framework library) and the Common Language Runtime which is the runtime in which .NET assemblies are run.
The dotNET framework includes many languages that use its libraries one of which is C#.
